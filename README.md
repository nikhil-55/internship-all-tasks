# internship-all-tasks

This repository contains all internship design tasks completed using Figma.

## 🔹 Task 1 – Job Portal UI
Design of a job search platform with job listing, filters, and application flow.

## 🔹 Task 2 – Bus Booking UI
Design of a bus booking system including seat selection and booking flow.

## 🔹 Task 3 – Event Booking UI
Design of a cross-platform (Web & Mobile) event booking system with seat selection and checkout.

## 🎯 Key Features
- User-centered design approach
- Clear booking flow
- Interactive seat selection
- Clean and minimal UI
- Cross-platform consistency

## 🛠 Tools Used
- Figma
